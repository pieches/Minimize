# Minimize

macOS 菜单栏常驻工具 — 鼠标滑到屏幕右上角右键，即可最小化或关闭窗口。

## 使用方式

1. 启动 App，菜单栏出现 `✕` 图标
2. 将鼠标移到屏幕右上角，会出现半透明高亮区域
3. 右键单击
   - **默认** → 最小化当前最上层窗口到 Dock
   - **设为「关闭」的 App** → 关闭窗口
   - **设为「不响应」的 App** → 无操作
4. 每次操作后自动聚焦到下一窗口，可连续操作

## 设置

菜单栏 → **设置...** 打开配置页，每个 App 有 3 种模式：

| 模式 | 效果 |
|------|------|
| 不响应 | 手势完全忽略 |
| 最小化 | 窗口收入 Dock（默认） |
| 关闭 | 关闭窗口 |

## 首次启动

首次运行且无任何配置时，自动弹出引导窗口，可快速为各 App 设置手势行为。

## 技术要点

- 纯 SwiftUI + AppKit，零第三方依赖
- 通过 Accessibility API 操控窗口（首次运行会弹出辅助功能授权提示）
- 通过 `CGWindowListCopyWindowInfo` 获取屏幕窗口 Z-order，自动跳过自身 UI
- 手势行为配置通过 `UserDefaults` JSON 持久化

## 构建

```bash
open macOSextension.xcodeproj

```
